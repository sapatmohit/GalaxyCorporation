# Galaxy Corporation - Global Agro Exporters

This is a modern, responsive website for Galaxy Corporation, an international trading and export company based in India. The company deals in pulses, legumes, rice, spices, oil seeds, coconuts, and fresh vegetables.

## Features

- Modern, corporate design with a clean white and navy-blue color palette
- Responsive layout optimized for both mobile and desktop
- Interactive components with smooth animations
- Product gallery with category filters
- Team member profiles with modal bios
- Contact form with Google Maps integration
- Animated counters for company statistics
- Affiliations and memberships showcase

## Technology Stack

- [Next.js 15](https://nextjs.org) with App Router
- [React 19](https://reactjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/) for animations

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
